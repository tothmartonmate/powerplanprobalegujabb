# Funkciók áttekintése

A PowerPlan dokumentáció funkciók fejezete a webalkalmazás fő szolgáltatásait mutatja be.

A dolgozat rendszere az alábbi fő csoportokra bontva magyarázza el a működést:

- felhasználói bejelentkezés és regisztráció
- egyéni kérdőív és javasolt étkezés/edzés
- dashboard az adatok és előrehaladás megjelenítésére
- edzéstervezés és étkezés-kezelés
- belső kommunikáció és üzenetek
