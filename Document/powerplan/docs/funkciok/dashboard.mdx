# Dashboard és előrehaladás követése

A felhasználói dashboard összegzi a legfontosabb információkat és az előrehaladást.

## Jellemzők

- napi állapot és edzéstervek
- üzenetek és értesítések
- adatvizualizáció a teljesítményről

A `dashboardController` az irányító felület mögött fut.
