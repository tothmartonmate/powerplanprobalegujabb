# Kommunikáció és üzenetek

A rendszer lehetővé teszi üzenetek küldését és fogadását a felhasználók között.

## Főbb elemek

- belső üzenetküldés
- admin és felhasználói kommunikáció
- értesítések a fontos eseményekről

A `messageController` kezeli az üzenetek folyamatát.
